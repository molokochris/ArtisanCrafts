# ArtisanCrafts — Full UX Deliverable (Tasks 2–8)

A complete interactive mid-fidelity → high-fidelity prototype built as a React app.
All task requirements are met within the single site, navigable via the top navbar.

## Pages

| Page           | Navbar link      | Task coverage                          |
|----------------|------------------|----------------------------------------|
| Home           | Explore          | Task 4, 5, 6, 8 — Home/Discovery       |
| Artisan Profile| Artisans         | Task 4, 5, 6, 8 — Artisan page         |
| Product Detail | (via cards)      | Task 4, 5, 6, 8 — Product Detail       |
| Shopping Cart  | 🛍 cart icon     | Task 4, 5, 6, 8 — Cart & Checkout      |
| Checkout       | (via Cart)       | Task 4, 5, 6, 8 — Checkout             |
| Style Guide    | Style Guide      | Task 7 — Colours, type, spacing, components |
| UX Flows       | UX Flows         | Task 2 — User Flow + Task Flow diagrams |
| Prioritization | Prioritization   | Task 3 — 2×2 matrix + 5 key insights   |

## Fidelity Switcher (top-centre pill)

- **Wireframe** — greyscale, crosshatch image placeholders, annotation panels visible
- **High-Fidelity** — warm artisan colour palette, real photography (Picsum), annotations hidden
- **Interactive** — same as High-Fi + hover/press micro-interactions on all buttons and cards

## Running locally

```
npm i
npm run dev
```
